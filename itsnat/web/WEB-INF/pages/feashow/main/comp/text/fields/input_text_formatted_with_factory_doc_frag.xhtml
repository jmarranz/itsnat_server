This example is an advanced use of an ItsNatHTMLInputTextFormatted component using a formatter factory. ItsNat provides a default formatter factory with several pluggable formatters (null formatter, display formatter, edit formatter and default formatter).

In proxy browsers like Opera Mini or UCWEB focus and blur are not fired, use a value with format mm/dd/yyyy if you change the current value.

In UCWEB, text controls do not fire events, when the client contacts the UCWEB server any change in text controls is automatically transmitted.
