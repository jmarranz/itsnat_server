The following example shows how we can embed SVG in an X/HTML document served with MIME text/html (the Feature Showcase uses this MIME). It works in MSIE 6+ with Adobe SVG Viewer (v3 or v6 beta) installed, and browsers with native SVG support (the <object> tag and <?import?> processing instruction are ignored, SVG code is rendered natively).

See the Reference Manual to know the limitations of this approach in MSIE.
