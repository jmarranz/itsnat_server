Modal layers in XUL are similar to HTML/XHTML, there are some significative differences: z-index is ignored because XUL does not have this property, by default a <panel> element is used and user markup should be included below this element.

In XUL only the first <panel> is modal, if you need two or more modal layers the "clean below" feature can be used to avoid the non-modal behavior of the second and successive layers.
