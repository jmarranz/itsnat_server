Any element group able to be an ElementList can be a free multiple selection list. Free multiple selection lists implement the interface ItsNatFreeListMultSel.

Basically free lists work the same as the <select> homologue, the main difference is that free lists need one child item element used as the list pattern and there is no default decoration of the selected items.

Free lists have in-place editing and support SHIFT and CTRL keys used to select/unselect elements.
