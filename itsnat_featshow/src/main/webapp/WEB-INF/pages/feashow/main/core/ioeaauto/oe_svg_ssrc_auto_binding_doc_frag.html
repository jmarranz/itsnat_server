This example is a variant of the previous one, in this case we need MSIE with Savarese Ssrc SVG/XUL or a browser with native SVG.

When a SVG document is loaded by Ssrc through an <object> or <embed> when the domain is the same, we have very limited access to the child document, most of properties of the child document are read-only. Fortunately we can change LocationURL indirectly calling Navigate(url) and using the #reference part of the URL we can "talk" to the child document and vice versa.

Savarese Ssrc SVG/XUL with <object> or <embed> needs absolute URLs, fortunately ItsNat auto-binding feature works with absolute URLs when the path of the parent document is the same.

With <iframe> there is no way to communicate parent and child documents in client because LocationURL is not defined.
