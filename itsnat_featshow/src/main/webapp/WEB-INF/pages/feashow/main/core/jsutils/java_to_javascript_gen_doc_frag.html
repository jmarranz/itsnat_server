This example shows how to construct JavaScript code to access a DOM client element using the server element and the ItsNat internal path system, how to transport a Java string as a JavaScript string, how to set an element property and to call an element method.

The utility class used is ScriptUtil obtained from the document calling ItsNatDocument.getScriptUtil().

The JavaScript generated by methods like ScriptUtil.getNodeReference is not documented and may change.

Note: this example does not work if the example tab is loaded directly for instance with this direct link. This is not a bug, ItsNat is designed to control form controls from server (this example changes a text box using custom JavaScript), in page loading time form controls are forced to be in sync with the server. Changing form controls with custom JavaScript in load time losing the sync with the server is not a very good practice, anyway to avoid this problem you can use a load event instead of the document load handler.
