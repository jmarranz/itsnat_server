W3C DOM Event Listeners are registered to listen browser DOM events, these client events are sent to the server and converted to Java W3C Events. The client DOM Element involved is the symmetric to the server DOM Element used to register the listener, ItsNat uses an identification system to locate both elements. This identification uses a smart client/server cache system faster than an XPath approach.

To register a DOM Event Listener use org.dom.events.EventTarget.addEventListener(String,EventListener,String) or ItsNatDocument.addEventListener(...) methods. By default EventTarget.addEventListener(String,EventListener,String) works in "remote" mode (ItsNatNode.isInternalMode() returns false).
