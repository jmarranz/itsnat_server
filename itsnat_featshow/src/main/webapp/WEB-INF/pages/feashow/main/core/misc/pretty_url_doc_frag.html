The standard way to request an ItsNat based new page includes the target servlet (ItsNat based) and an itsnat_doc_name with the template used to build the new page. This URL is not "pretty", ItsNat does not provide an "out of the box" pretty URL pattern but it offers some support to build the preferred style.

When a new page request is received by an ItsNat based servlet (not an event), ItsNat ever executes the global ItsNatServletRequestListener registered, calling ItsNatServlet.addItsNatServletRequestListener, if any. If the request does not include the itsnat_doc_name parameter (or attribute) there is no ItsNatDocument created. This global listener may detect this scenario and analyze the URL used (for instance a pretty URL) to build an itsnat_doc_name attribute to set to the request, and forward again the request and response objects to the ItsNat servlet. ItsNat looks for itsnat_doc_name first as an attribute if not found the as a request parameter.
